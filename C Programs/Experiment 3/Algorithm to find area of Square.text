Algorithm to find area of Square.

1. START
2. DECLARE VARIABLES `side` AND `area` AS FLOAT
3. CLEAR THE SCREEN
4. DISPLAY "ENTER SIDE OF SQUARE:-"
5. READ THE INPUT VALUE INTO VARIABLE `side`
6. CALCULATE `area` AS `side * side`
7. DISPLAY "AREA IS" FOLLOWED BY `area`
8. WAIT FOR USER INPUT TO CLOSE THE PROGRAM
9. END
