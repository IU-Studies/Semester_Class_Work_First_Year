Algorithm to find area of Circle.

1. START
2. DECLARE VARIABLES `radius` AND `area` AS FLOAT
3. CLEAR THE SCREEN
4. DISPLAY "ENTER RADIUS OF CIRCLE:-"
5. READ THE INPUT VALUE INTO VARIABLE `radius`
6. CALCULATE `area` AS `3.14 * radius * radius`
7. DISPLAY "AREA IS" FOLLOWED BY `area`
8. WAIT FOR USER INPUT TO CLOSE THE PROGRAM
9. END
