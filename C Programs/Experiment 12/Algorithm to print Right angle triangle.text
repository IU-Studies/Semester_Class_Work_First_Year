Algorithm to print Right angle triangle.

1. START
2. DECLARE VARIABLES `user`, `i`, AND `j` AS INTEGERS
3. CLEAR THE SCREEN
4. DISPLAY "ENTER NUMBER OF ROWS:-"
5. READ THE INPUT VALUE INTO VARIABLE `user`
6. FOR EACH ROW `i` FROM 1 TO `user`, INCREMENTING BY 1:
- a. FOR EACH COLUMN `j` FROM 1 TO `i`, INCREMENTING BY 1:
- i. PRINT AN ASTERISK `*` FOLLOWED BY A SPACE
- b. MOVE TO THE NEXT LINE
7. WAIT FOR USER INPUT TO CLOSE THE PROGRAM
8. END
